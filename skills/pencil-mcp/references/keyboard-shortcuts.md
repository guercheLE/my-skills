# Pencil Keyboard Shortcuts

> **Source:** https://docs.pencil.dev/core-concepts/keyboard-shortcuts

---

## AI Chat

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Toggle AI chat |
| `Cmd/Ctrl + T` | New chat tab |
| `Cmd/Ctrl + W` | Close current chat tab |
| `Ctrl + Tab` | Next chat tab |
| `Ctrl + Shift + Tab` | Previous chat tab |
| `↑` / `↓` | Prompt history |

## Tools

| Shortcut | Action |
|----------|--------|
| `V` | Move tool |
| `H` | Hand tool |
| `R` | Rectangle tool |
| `O` | Ellipse tool |
| `A` / `F` | Frame tool |
| `T` | Text tool |
| `N` | Sticky note tool |

## Editing

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + C` | Copy |
| `Cmd/Ctrl + V` | Paste |
| `Cmd/Ctrl + D` | Duplicate selection |
| `Cmd/Ctrl + X` | Cut |
| `Delete / Backspace` | Delete selection |
| `Arrow keys` | Move selection by 1px (or swap order in flex layout) |
| `Shift + Arrow keys` | Move selection by 10px |
| `Cmd/Ctrl + G` | Group |
| `Cmd/Ctrl + Shift + G` | Ungroup |
| `Cmd/Ctrl + Option/Alt + G` | Apply flex layout to selection |
| `Cmd/Ctrl + [` | Send backward |
| `Cmd/Ctrl + ]` | Bring forward |
| `[` | Send to back |
| `]` | Bring to front |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |

## Selection

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + A` | Select all |
| `Cmd/Ctrl + Click` | Direct select (deepest element) |
| `Shift + Click` | Add to selection |
| `Shift + Enter` | Select parent |
| `Enter` | Select children |
| `Esc` | Clear selection |

## Components

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Option/Alt + K` | Toggle: convert to component / convert back to standard element |
| `Cmd/Ctrl + Option/Alt + X` | Detach component instance |

## Canvas Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Scroll` | Zoom |
| `Spacebar + Drag` | Pan canvas |
| `Shift + Scroll` | Horizontal pan |
| `=` | Zoom in |
| `-` | Zoom out |
| `0` | Zoom to 100% |
| `1` | Zoom to fit all |
| `2` | Zoom to selection |

## Canvas Settings

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + '` | Toggle pixel grid |
| `Cmd/Ctrl + Shift + '` | Toggle snap to pixel grid |

## File Operations

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + S` | Save file |
| `Cmd/Ctrl + N` | New file |
| `Cmd/Ctrl + O` | Open file |
