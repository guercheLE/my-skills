# my-skills

A collection of custom agent skills for GitHub Copilot, focused on UI/UX design with [Pencil](https://pencil.dev) and skill quality auditing.

## Skills

### [pencil-mcp](skills/pencil-mcp/)

Expert guide for creating, editing, and validating UI designs in `.pen` files using the Pencil MCP server — and for translating those designs into production-ready code for any platform (React, SwiftUI, Compose, XAML, etc.).

**References:** setup & troubleshooting, `.pen` format spec, keyboard shortcuts, platform-specific guides (web, iOS/macOS, Android, Windows, Linux).

### [pencil-uiux-design](skills/pencil-uiux-design/)

Comprehensive UI/UX design guide for building professional, accessible interfaces in Pencil. Covers design systems, component patterns, accessibility standards, and platform guidelines.

**References:** component patterns, accessibility, platform guidelines.

### [skill-similarity-reliability-audit-bestpractices](skills/skill-similarity-reliability-audit-bestpractices/)

Audits the reliability of skill similarity and deduplication findings by validating YAML frontmatter, scoring metadata quality, and quantifying confidence tiers for duplicate detection. Designed for repos with hundreds or thousands of skills.
